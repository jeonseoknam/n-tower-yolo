# N-Seoul Tower Detect program
- 숭실대학교 컴퓨터학부 컴퓨터비전 수업 과제(송현주 교수님)

# Outline
- 입력 이미지가 입력되면, 이미지 내부에 N-Seoul Tower 객체 여부를 판단하는 프로그램 작성
- N-Seoul Tower가 있다면 Red Bounding Box로 결과 이미지를 생성(result.jpg)
- N-Seoul Tower가 없다면 None 출력
